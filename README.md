Express Blog Website

This project is a full-stack blog application built using Node.js, Express, and EJS that allows users to create, edit, view, and delete blog posts. The application follows a RESTful API architecture, where a backend API handles blog post data while a separate server renders the user interface and communicates with the API using HTTP requests.

The API server manages blog posts stored in an in-memory data structure and exposes endpoints for retrieving all posts, viewing a single post, creating new posts, updating existing posts, and deleting posts. The frontend server uses Axios to interact with the API and dynamically renders pages using EJS templates.

Key features include:

Creating new blog posts

Viewing all blog posts on the homepage

Editing existing posts

Deleting posts

RESTful API endpoints using Express

Dynamic server-side rendering with EJS

Communication between services using Axios

This project demonstrates the implementation of CRUD operations, REST API design, server-side rendering, middleware usage, and separation of frontend and backend logic in a Node.js environment.
